# Competitor Matrix — Channel Models and Player Types

| Player type | Main channel | Main value proposition | Strengths | Weaknesses |
|---|---|---|---|---|
| Marketplace seller | WB / Ozon / marketplaces | price + speed + breadth | traffic, scale, fast testing | weak brand moat |
| D2C furniture brand | site + ads + socials | curated offer + trust | brand control, better margin | CAC pressure |
| Salon / showroom | offline + WhatsApp / site | service + touch + project help | trust, upsell, project sales | limited scale |
| Local workshop | referrals + Instagram/Telegram/site | customization | fit, revision speed | weak systems |
| Complectation studio | designers + projects | turnkey furnishing | high ticket, bundled sales | complex operations |
| Vintage/antique dealer | showroom + social + marketplace | uniqueness, story | scarcity, premium feel | inventory instability |
| Importer / sourcing operator | B2B + trade relationships | access to factories / China | sourcing breadth | compliance/logistics risk |
| Premium gallery / designer dealer | showroom + private clients | originals / collectible | status, PR | niche demand |
