# Market Map — Furniture / Decor / Home Goods / Home Tech

## Executive view
Рынок мебели и home-группы в России и Москве в 2025–2026 годах характеризуется тремя сильными сдвигами:

1. **Digital-first demand:** большая часть мебельного спроса уже проходит онлайн; по данным Forbes/T-Data, в 2025 году **79% мебели в России было куплено через интернет**. [Confirmed Fact; source: SRC-001]
2. **Category polarization:** наиболее быстро росли понятные и массовые группы — **садовая мебель (+96%), мебель для сна (+85%), мягкая мебель (+80%)** на Wildberries. [Confirmed Fact; source: SRC-002]
3. **Premium and collectible split:** параллельно растет спрос на винтаж, collectible design и проектную комплектацию high-end интерьеров; продажи винтажной мебели и декора в I кв. 2025 выросли на **27% год к году**, а за три года — более чем в **2,5 раза**. [Confirmed Fact; source: SRC-003]

## Market architecture

### A. Mass online / marketplace-driven
Игроки:
- маркетплейсы
- массовые бренды
- D2C-производители
- продавцы на Ozon / Wildberries / Avito

Core demand:
- кровати
- матрасы
- диваны
- шкафы
- стеллажи
- outdoor
- стандартные столы/стулья
- дешевый декор и хранение

Best metrics:
- высокий поисковый спрос
- простота выбора по фото
- высокая контентная визуальность
- умеренная логистическая сложность
- быстрый оборот

### B. Project complectation / design-driven middle
Игроки:
- студии комплектации
- салоны мебели
- локальные производства
- интеграторы свет+мебель+декор

Core demand:
- кухни
- встроенная мебель
- гардеробные
- dining groups
- lighting
- mirrors
- textiles
- premium basic furniture

Best metrics:
- высокий средний чек
- пакетируемость
- повторяемость в проектах
- высокий LTV через сервис

### C. Premium / designer / collectible
Игроки:
- showroom-операторы
- авторские бренды
- галереи
- дилеры международных марок
- кураторы комплектации

Core demand:
- statement seating
- sculptural tables
- collectible decor
- premium lighting
- object furniture
- limited editions

Best metrics:
- высокая маржа
- PR / status / identity value
- эффект портфолио
- низкая commoditization

### D. Vintage / antique / resale
Игроки:
- антикварные галереи
- ресейл-площадки
- кураторы винтажа
- реставрационные мастерские

Core demand:
- люстры / лампы
- картины / posters / objets
- кресла
- комоды
- буфеты
- зеркала
- tables / side tables

Best metrics:
- storytelling power
- scarcity
- content virality
- suitability for premium eclectic interiors

Игроки:
- импортеры
- white-label sellers
- ODM/OEM-сборщики ассортимента
- закупочные агенты
- логистические партнеры

Core demand:
- lighting
- decor
- outdoor
- chairs
- side tables
- accessories
- hardware-intensive furniture
- branded OEM series

Best metrics:
- ширина матрицы
- скорость теста ассортимента
- стоимость unit economics
- supply flexibility

Игроки:

- cabinetry workshops

Core demand:
- кухни
- шкафы
- встроенные системы

- upholstered beds
