# Furniture Categories

1.  **Sleep furniture**: Кровати, матрасы, основания.
2.  **Upholstered furniture**: Диваны, кресла, пуфы.
3.  **Storage**: Шкафы, комоды, стеллажи.
4.  **Cabinetry**: Кухни, встроенная мебель.
5.  **Small Tables**: Кофейные, журнальные столики.
6.  **Outdoor**: Мебель для террас и садов.
