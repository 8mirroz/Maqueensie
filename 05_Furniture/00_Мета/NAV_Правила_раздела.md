# NAV — Правила раздела

## Схема именования
`ABBR_короткое_русское_название.md`

Принятые аббревиатуры:
- `RYN` рынок
- `SPR` спрос
- `CAT` категории/ассортимент
- `SUP` поставки
- `GTM` офферы и go-to-market
- `MRK` маркетинг
- `SRC` реестр источников
- `CMP` конкуренты
- `PRC` ценообразование
- `TPL` шаблоны
- `GID` гайды
- `NAV` навигация
- `DEV` roadmap развития
- `BKL` backlog гипотез
- `KPI` метрики обновления

## Принцип развития раздела
- Новые заметки создаются в соответствующей ветке (`01_Рынок`..`06_Маркетинг`, `90_Развитие`).
- Для аналитики используются явные ссылки на [[SRC_мебель_реестр_источников]].
- При добавлении новой ветки сначала обновляется [[NAV_Карта_раздела]].
