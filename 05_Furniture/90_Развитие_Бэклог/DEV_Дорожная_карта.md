# DEV — Дорожная карта развития раздела

## Wave 1 — усиление данных
- Заполнить `PRC` фактическими диапазонами цен по Tier A категориям.
- Добавить 15-30 проверенных datapoints на категорию.
- Зафиксировать источники и даты обновления.

## Wave 2 — углубление конкурентов
- Расширить `CMP` до сегментов (mass / project / premium / vintage / import).
- Создать карточки топ-игроков и связать с `GTM`.

## Wave 3 — связка спроса и контента
- Для каждого Tier A сегмента связать `SPR -> GTM -> MRK`.
- Выделить пакеты офферов по аудиториям.

## Wave 4 — операционный цикл обновления
- Ввести ежемесячный cycle review по KPI.
- Архивировать устаревшие гипотезы из `BKL`.
