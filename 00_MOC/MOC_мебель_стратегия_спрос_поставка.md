# MOC — Furniture / Decor / Home Goods Demand-Supply Strategy

## Purpose
Это мастер-узел для market intelligence по мебели, декору, home goods, lighting, smart-home и adjacent-сегментам для России и Москвы.

## Strategic framing
Рынок нужно вести не как один каталог, а как несколько параллельных бизнес-машин:

1. **Mass marketplace machine**
2. **Project complectation machine**
3. **Premium design / collectible machine**
4. **Vintage / antique / resale machine**
5. **China-linked sourcing / OEM machine**
6. **Local workshop / custom production machine**

## Highest-priority demand zones
- мебель для сна
- мягкая мебель
- хранение / шкафы / встроенная мебель
- кухни / cabinetry
- outdoor / terrace furniture
- lighting
- decor / mirrors / textiles / styled objects
- vintage / collectible decor
- smart-home / light-control / premium home tech

## Recommended business use
- консалтинг: сегментация, pricing, канал-стратегия
- маркетинг: контент-матрицы, offer-design, demand capture
- продажи: пакетирование, tiering, assortment architecture
- закупки: supply mix China/local/collectible
- knowledge base: standard cards, competitor scans, pricing tables
